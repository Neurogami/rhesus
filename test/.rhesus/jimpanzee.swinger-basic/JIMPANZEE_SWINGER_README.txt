You need to provide the Swinger project files under <%= swinger_submodule_dir %>.

The suggested way, if you are using git, is to add the code as a git submodule:


   git submodule add git://github.com/Neurogami/swinger.git  <%= swinger_submodule_dir %>/swinger

   git add <%= swinger_submodule_dir %>/swinger
   
   git commit -m "Adding the super bad Neurogami fork of Swinger to <%= swinger_submodule_dir %>/swinger as a submodule"


See http://stuff.lilleaas.net/git_submodules  for more info.
